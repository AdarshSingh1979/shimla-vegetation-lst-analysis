## Vegetation Change and Land Surface Temperature Analysis of Shimla (2015–2025)

## Overview
This project analyzes vegetation change and land surface temperature (LST) patterns in Shimla district, Himachal Pradesh using Landsat 8 satellite imagery.
The study compares satellite data from 2015 and 2025 to understand how vegetation dynamics influence surface temperature.

## Tools Used
1. QGIS 3.40
2. Landsat 8 Satellite Data (USGS EarthExplorer)
3. Raster Analysis
4. NDVI Calculation
5. Land Surface Temperature Derivation

## Methodology
1. Download Landsat 8 satellite data
2. Calculate NDVI using Red and NIR bands
3. Derive Land Surface Temperature from thermal band
4. Perform vegetation change detection
5. Analyze NDVI–LST spatial relationship

## Key Results
1. Vegetation Loss: ~182 km²
2. Vegetation Gain: ~70 km²
3. Net Vegetation Decline: ~112 km²
4. Areas with vegetation loss show higher surface temperatures

## Data Source
USGS EarthExplorer – Landsat 8 OLI/TIRS
